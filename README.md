RohitCRO — Static GitHub Pages site
Files:
- index.html
- style.css
- CNAME (contains rohitcro.com)

To deploy:
1) Create a repository named `rohitcro.github.io` on GitHub (you already did).
2) Upload these files to the repository root (via web upload or git push).
3) In the repository's Settings → Pages, ensure branch is main (or master) and folder is root.
4) GitHub Pages will publish at https://rohitcro.github.io within minutes.
5) To use your custom domain (rohitcro.com), add these DNS records at Hostinger DNS panel:
   - A 185.199.108.153
   - A 185.199.109.153
   - A 185.199.110.153
   - A 185.199.111.153
   - CNAME www rohitcro.github.io
   Also add a file named CNAME with 'rohitcro.com' in the repo (included here).
